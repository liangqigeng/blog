世界互联网颁奖晚会，我们获奖了…

2018.03.21 - 卖砖头的小女孩

最近有网友问我，倒不如你写一篇“宣传柚段子”的软文，让我看你平时是装逼还是真牛逼。所以今天冒昧在这里来个抛砖引玉，请大家多多指教…

打假多了，总会有些无聊的人想报复，这也是没办法啊…

晚会正式开始，感谢往届获奖的大V们对互联网发展作出的奉献，他们分别是：“骂晕，你眼红”。还有今天新晋小V：卖砖头的小女孩。有请他们上台…

1圆桌会议

第一环节圆桌会议，问题我早就想好了。

主持 : 电商平台假货横行，商家无道，买到假货，究竟谁负责。

骂晕 ：我们只负责平台管理，尽量杜绝假货，买到假货找卖家。

李眼红 ：前两天我在某宝买了一盒Condom，用后发觉那窟窿比手指还大，这能叫安全套么，简直就是个圈套。现在小三有了，我能找卖家负责么，你妹的。

骂晕：那前两天，我也在某度搜索灰指甲治疗，竟然出来的是美甲店的广告信息，治灰指甲去美甲店，那把自已往死里推。这样的医疗广告一定要禁止，这是生命的绑架，我说得对吗，你总？

你眼红 ：我不做，其它人也会做，我只能尽量做到那信息更加可信 …

骂晕：那不是把假做得更加真实。

李眼红 ：TMD，你是故意曲解我的意思么 …

骂晕：喂则西事件是不是你干的…

李眼红 ：你闭嘴!

骂晕：你回答我，是不是你干的…

主持 ：大家冷静，我们马上进入第二环节。

2为获奖新晋颁奖

主持：现在有请“骂总，你总”为新晋颁奖，并请获奖者讲话。

卖砖头的小女孩：现在这年头，搞游戏的误人子弟，搞电商的滥竽充数，搞搜索的草菅人命。唯独搞内容创作的“柚段子”，在提高内容质量同时，不断提高服务水平，目前持续受到网络各界人士的好评骚扰。

骂总 : 原来这是你的产品，怎么这么巧，“柚段子公众号”我早已关注了。

你总 : 我们公司好几个网站也是用“柚段子”的模板做的，用户粘贴度非常高。

主持 : 看来这个新晋奖真的是实至名归，也许今天“骂总和你总”的成就，也是源于“柚段子”。

骂总 : 你TMD的，扯蛋…

台下观众 : 快把柚段子的二维码发出来，我也要关注 …

一个好的项目产品，就是质量与服务的集合；

一篇好的营销软文，就是一个走心的故事和段子。

作者：卖砖头的小女孩，男扮女装，Web前端开发攻城狮，Adobe认证设计师，互联网运营策划师，国家指定低保低收入人士。曾就职于某易，之后一直堕落，现网上终日行乞……
